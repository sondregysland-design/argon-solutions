## Claude Code opplæring for olje- og gassindustrien

AI-assistert koding revolusjonerer softwareutvikling. **Claude Code** fra Anthropic er et av de kraftigste verktøyene tilgjengelig, og Argon Solutions tilbyr skreddersydde workshops for utviklingsteam i energisektoren.

## Hva er Claude Code?

Claude Code er en AI-drevet kodingsassistent som kjører direkte i terminalen. Den kan lese kodebasen din, skrive kode, kjøre tester, og utføre komplekse utviklingsoppgaver autonomt. Argon Solutions bruker Claude Code daglig i all vår utvikling.

## Hva lærer du på workshopen?

Argon Solutions sin Claude Code-workshop dekker:

### Dag 1: Grunnleggende
- Installasjon og konfigurasjon
- Grunnleggende kommandoer og arbeidsflyter
- Kontekststyring og CLAUDE.md-filer
- Best practices for prompting

### Dag 2: Avansert
- Custom skills og hooks
- MCP-servere for integrasjon med interne systemer
- Subagenter og parallell utvikling
- CI/CD-integrasjon med Claude Code

### Dag 3: Bransjespesifikt
- Tilpasning for olje og gass-prosjekter
- Sikkerhet og compliance-hensyn
- Integrasjon med eksisterende utviklingsprosesser
- Hands-on prosjekt med deres egen kodebase

## Hvem passer workshopen for?

- Utviklingsteam i olje- og gasselskaper
- IT-avdelinger som ønsker å akselerere utvikling
- Tekniske ledere som vurderer AI-verktøy
- Konsulenter som jobber med energisektoren

- **Praktisk erfaring** — Vi bruker Claude Code daglig, ikke bare i demo
